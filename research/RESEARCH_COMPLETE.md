## 🎓 Lessons Learned

### What Works Well

1. Streaming responses provide excellent UX
2. Tool calling is intuitive for LLMs
3. Approval gates balance safety/autonomy
4. Type safety catches errors early
5. Vercel AI SDK is well-designed

### Common Pitfalls (Documented)

1. Don't await addToolResult (deadlock)
2. Always use tool() helper (not raw objects)
3. Set reasonable maxSteps (not Infinity)
4. Return errors, don't throw them
5. Manage context window (trim messages)
6. Write clear tool descriptions
